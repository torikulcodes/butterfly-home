# Frontend Developer Task – SM Technology

This project is a **replica of the reference design** provided by SM Technology, created using **HTML, CSS, JavaScript, and React**. The website is fully responsive, pixel-perfect, and includes animations as requested.

## 🔗 Live Demo
Check the live version here: [Live Site](https://lovely-tulumba-9b8a16.netlify.app/)

## 🛠 Technologies Used
- HTML5
- CSS3,Tailwind
- JavaScript (ES6+)
- React
- Framer motion
- Gsap

## 🎨 Features
- Pixel-perfect design replication of the reference site.
- Fully responsive layout (mobile, tablet, desktop).
- Smooth animations for banners and other elements.
- Clean and well-structured code.
